We are announcing a new Crystal release 1.20.2 with a couple of regressions fixed.

See [the release notes of 1.20.0](/2026/04/16/1.20.0-released/) for all the
changes introduced in Crystal 1.20.

Pre-built packages are available on [GitHub
Releases](https://github.com/crystal-lang/crystal/releases/tag/1.20.2) and our
official distribution channels. See
[crystal-lang.org/install](https://crystal-lang.org/install/) for installation
instructions.

## Stats

This release includes [2 bug fixes since
1.20.1](https://github.com/crystal-lang/crystal/pulls?q=is%3Apr+milestone%3A1.20.2)
by 1 contributor. We thank all the contributors for all the effort put into
improving the language! ❤️

## Changes

### Randomness in `Range#sample`

This patch release brings the regression fix for `Range#sample` ([#16909]) that could
eventually lose randomness to the 1.20 release branch.
This was previously released as a backport for 1.19 in Crystal 1.19.2 ([#16853]).

_Thanks, [@ysbaddaden]_
