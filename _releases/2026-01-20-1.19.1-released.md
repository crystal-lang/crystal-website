We are announcing a new Crystal release 1.19.1 with two regressions fixed.

See [the release notes of 1.19.0](/2026/01/15/1.19.0-released/) for all the changes introduced in Crystal 1.19.

Pre-built packages are available on [GitHub
Releases](https://github.com/crystal-lang/crystal/releases/tag/1.19.1) and our
official distribution channels. See
[crystal-lang.org/install](https://crystal-lang.org/install/) for installation
instructions.

## Stats

This release includes [2 changes since
1.19.0](https://github.com/crystal-lang/crystal/pulls?q=is%3Apr+milestone%3A1.19.1)
by 1 contributor.  We thank all the contributors for all the effort put into
improving the language! ❤️

## Changes

This patch release fixes regressions in timer conversions from absolute to relative
time ([#16581], [#16583]) — both introduced in 1.19.0.

For more details, visit the [full
changelog](https://github.com/crystal-lang/crystal/releases/tag/1.19.1).

[#16581]: https://github.com/crystal-lang/crystal/pull/16581
[#16583]: https://github.com/crystal-lang/crystal/pull/16583
