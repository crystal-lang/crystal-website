We are announcing a new Crystal release 1.20.1 with a couple of regressions fixed.

See [the release notes of 1.20.0](/2026/04/16/1.20.0-released/) for all the
changes introduced in Crystal 1.20.

Pre-built packages are available on [GitHub
Releases](https://github.com/crystal-lang/crystal/releases/tag/1.20.1) and our
official distribution channels. See
[crystal-lang.org/install](https://crystal-lang.org/install/) for installation
instructions.

## Stats

This release includes [9 changes since
1.20.0](https://github.com/crystal-lang/crystal/pulls?q=is%3Apr+milestone%3A1.20.1)
by 2 contributors.  We thank all the contributors for all the effort put into
improving the language! ❤️

## Changes

### Kernel TLS

Support for Kernel TLS was introduced in 1.20.0 but it showed issues in production. Some have been fixed ([#16888]) but there are still some problems so it was disabled by default ([#16897]).

You can opt-in to Kernel TLS by adding the `ENABLE_KTLS` option:

```cr
ssl_context.add_options(OpenSSL::SSL::Options::ENABLE_KTLS)
```

_Thanks, [@ysbaddaden]_

[#16888]: https://github.com/crystal-lang/crystal/issues/16888
[#16897]: https://github.com/crystal-lang/crystal/issues/16897

### Randomness in `Range#sample`

This patch release fixes a regression in `Range#sample` ([#16853]) that could
eventually lose randomness.

_Thanks, [@ysbaddaden]_

[#16853]: https://github.com/crystal-lang/crystal/issues/16853
