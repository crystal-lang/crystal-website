Linux

Many Linux distribution have Crystal available in their system packages. It might be some older version though. Third party package managers can be more up to date. DEB and RPM packages of the most recent release are available in our own package repository.

Crystal Official

Installer (DEB, RPM)
curl -fsSL https://crystal-lang.org/install.sh | sudo bash
instructions

The installer automatically adds the respective package repository at OBS for distributions with DEB or RPM packages. You can also install these packages manually

Tarballs are available on the Releases page on GitHub.
